#Stock Portfolio Tracker (Python)

A simple Python-based **Stock Portfolio Tracker** that allows users to calculate their total investment using predefined stock prices.  
This project demonstrates core Python concepts like dictionaries, loops, conditionals, file handling, and CSV export.

---

## Project Objective

To build a beginner-friendly stock tracking application where:
- Users enter stock names and quantities
- Prices are fetched from a predefined dictionary
- Total investment is calculated
- Results are saved to files for future reference
